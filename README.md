# MineralTree
Mineral Tree of HawnHan

    目前允许通过种植得到的矿物（星号为如果装有该类mod则可以进行种植）
    -钢铁
    -白银
    -黄金
    -翡翠
    -零部件树
    -玻璃钢树
    -高级零部件树
    -钛合金树*（闪耀科技/Glitter Tech）
    -阿尔法树*（闪耀科技/Glitter Tech）
    -贝塔树*（闪耀科技/Glitter Tech）

    原作者：jxb12306（原先的四种树图标都是用的原作者的）

    添加了新的研究标签，在解锁“开启矿物质树”后可以任意解锁其余树的研究，未来可能添加更多矿物。
    嘛也别问我为啥不改后面树的图标，我PS贼烂！
    该MOD已经被我大修，原作者仅是单纯的添加了四种矿物质树
    我在其基础上定义了父系（这个mod现在变得十分容易修改和添加新的作物，我相信任何人都可以做到），对植物的部分属性进行修改（正义方向），改为可室内，无光生长，可水栽培，收获后无需重新种植，简单添加新的图标。
    大修之后其实跟原作品已有十分大的区别，代码更加简洁，载入速度应有提升（但我并未测试）
    因此mod作者是我了现在
